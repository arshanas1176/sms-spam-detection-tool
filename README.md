# SMS Spam Detection Tool

## Overview

SMS Spam Detection Tool is a machine learning web application that classifies SMS messages as Spam or Legitimate using TF-IDF Vectorization and Logistic Regression.

## Features

* Real-Time SMS Prediction
* Bulk CSV Prediction
* Prediction History
* Dashboard Analytics
* Confidence Scores
* Downloadable Reports

## Technologies Used

* Python
* Streamlit
* Scikit-Learn
* TF-IDF Vectorizer
* Logistic Regression
* Plotly
